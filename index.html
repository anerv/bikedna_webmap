Bicycle Infrastructure in Denmark

This map provides an overview of data on dedicated bicycle infrastructure in Denmark based on two data sources: OpenStreetMap (OSM) and GeoDanmark. Included layers are:
  • All dedicated bicycle infrastructure.
  • An overview of corresponding infrastructure in the two data sources.
  • Disconnected components.
  • The largest connected components in the OSM and GeoDamark data.
The results have been produced using the open source tool for quality assessment of bike infrastructure data BikeDNA. See this site for exact queries used to extract the OSM data.

For further details on the map content, see the descriptions in the map legend.

The analysis has been funded by the Danish Road Directorate.
